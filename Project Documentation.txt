The scope of this project includes:

-Providing admission-related information for different programs

-Displaying eligibility criteria and required documents

-Showing application deadlines

-Offering a text-based chatbot interface

-Supporting easy database updates for new universities/programs

This bot is designed mainly for students who are planning to apply to universities and need quick, reliable, and structured admission guidance.

5. System Overview

The University Admissions Bot is a Python-based chatbot that interacts with users through a command-line interface. It reads admission data from a structured database (JSON file) and provides relevant responses based on user queries.

Chatbots in the admission domain help automate repetitive queries and provide instant responses, improving efficiency and accessibility for students.
